# 🍷 Red Wine Quality - Descriptive Statistics Analysis

## 📌 Overview

This project is part of my early journey in a AI ML bootcamp, focusing on descriptive statistics to understand the characteristics of a red wine dataset and explore relationships between features and wine quality.

## 🎯 Objectives

* Understand the distribution of each feature
* Identify patterns and key characteristics in the dataset
* Explore initial relationships between features and wine quality
* Practice Exploratory Data Analysis (EDA)

## 📂 Dataset

The dataset contains several chemical properties of red wine, including:

* Fixed acidity
* Volatile acidity
* Citric acid
* Residual sugar
* Chlorides
* Alcohol
* pH
* Sulphates
* Quality (target variable)

## 🛠 Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Google Colab

## 🔍 Key Insights

* Most wines fall within the **quality range of 5–6**, indicating an imbalanced distribution centered around mid-quality wines.
* **Alcohol** shows a tendency of positive correlation with wine quality — higher alcohol content is often associated with better quality.
* Higher **volatile acidity** tends to be associated with lower wine quality.
* Some features show non-normal distributions, which may require additional consideration for future modeling.

## 📊 Analysis Process

1. Data Understanding
2. Data Cleaning (handling missing values if any)
3. Descriptive Statistics
4. Data Visualization
5. Insight Extraction
